# Solstic Calendar Web App

## Development

Install https://pnpm.io/installation then `pnpm install`.

Run development server: `pnpm run dev` and goto http://localhost:5173

Development server automatically reloads browser with having to hit refresh or press `CTRL+R`!

## Recommended IDE Setup

[VS Code](https://code.visualstudio.com/) + [Svelte](https://marketplace.visualstudio.com/items?itemName=svelte.svelte-vscode).
